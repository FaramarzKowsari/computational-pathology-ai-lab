# Release v1.0.0

1. Ensure CI is green and `reports/TEST_REPORT.md` matches the current commit.
2. Run `python scripts/create_checksums.py`.
3. Commit the checksum manifest.
4. Open GitHub **Releases > Draft a new release**.
5. Create tag `v1.0.0` from `main`.
6. Title: `Computational Pathology AI Lab v1.0.0`.
7. Summarize executable components and explicitly state that real and clinical benchmarks are not included.
8. Attach the source ZIP generated by GitHub and, optionally, the companion sample PDF.
9. Publish the release.
10. After Zenodo archives the release, replace the software DOI placeholders in README, CITATION, website, and metadata in a follow-up patch release.
