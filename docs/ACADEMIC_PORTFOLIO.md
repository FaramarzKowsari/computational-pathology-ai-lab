# Academic and Employment Portfolio Positioning

This repository is designed to support four evidence-based portfolio narratives:

- **Medical AI researcher:** research questions, leakage controls, calibration, uncertainty, domain shifts, and validation planning.
- **Machine-learning engineer:** package design, configuration, tests, API, container, CI, DVC, MLflow hooks, and monitoring.
- **Computer-vision engineer:** patch pipelines, CNNs, transfer learning, ViT, WSI utilities, explainability, and MIL.
- **Instructor and technical author:** teaching notebooks, syllabus, methodology, model/data cards, technical report, website, and companion book.

Claims in a CV should refer only to features actually implemented or experiments actually executed. Use "designed an evaluation-ready adapter" rather than "outperformed foundation models" until benchmark evidence exists.
