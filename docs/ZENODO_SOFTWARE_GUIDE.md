# Zenodo Guide for the Software DOI

The book DOI is already reserved: `10.5281/zenodo.21444837`. Do not reuse that DOI for software.

For the repository software:

1. Sign in to Zenodo and connect the GitHub account.
2. Enable `computational-pathology-ai-lab` in the GitHub integration.
3. Publish GitHub release `v1.0.0`.
4. Wait for the new Zenodo software record.
5. Edit metadata:
   - Resource type: Software
   - Title: Computational Pathology AI Lab
   - Version: 1.0.0
   - Creator: Faramarz Kowsari
   - ORCID: 0000-0003-1692-0453
   - License: MIT
   - Related identifier: book DOI `10.5281/zenodo.21444837`, relation `is supplemented by` or the relation available in the interface that best represents the companion book.
6. Publish the software record.
7. Replace every `Software DOI: To be assigned` placeholder with the minted DOI.
8. Keep the concept DOI and version DOI distinct in documentation.
