# Google Search Console Guide

1. Deploy GitHub Pages and verify that the canonical site opens.
2. Add the URL-prefix property `https://FaramarzKowsari.github.io/computational-pathology-ai-lab/` to Search Console.
3. Use the HTML-file verification method if offered. Put the verification file inside `website/`, commit, and redeploy.
4. Submit `https://FaramarzKowsari.github.io/computational-pathology-ai-lab/sitemap.xml`.
5. Request indexing for the home page and `book.html`.
6. Keep the repository name, page title, author identity, DOI, ORCID, and book metadata consistent.
7. Do not create doorway pages or repeated keyword blocks. Improve discoverability through substantive documentation and releases.
