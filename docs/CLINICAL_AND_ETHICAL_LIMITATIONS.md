# Clinical and Ethical Limitations

This repository is not a medical device. It has not been validated for diagnosis, triage, prognosis, treatment planning, or deployment in a clinical workflow.

Potential failure sources include staining and scanner shift, tissue preparation artifacts, label noise, sampling bias, demographic imbalance, spurious visual shortcuts, overconfidence, and leakage between patches from the same patient.

Any future translational study requires governance, privacy review, domain experts, representative data, external validation, prospective evaluation, human-factors analysis, security review, and compliance with applicable regulation.
