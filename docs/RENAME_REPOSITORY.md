# Rename the Repository

1. Open the repository on GitHub.
2. Select **Settings**.
3. Under **General**, find **Repository name**.
4. Change it to `computational-pathology-ai-lab` and confirm.
5. Update the About description to: `Production-oriented computational pathology research platform for cancer detection, explainable AI, uncertainty, whole-slide learning, and MLOps.`
6. Add topics: `computational-pathology`, `medical-ai`, `histopathology`, `computer-vision`, `deep-learning`, `pytorch`, `mlops`, `explainable-ai`, `whole-slide-image`, `research-software`.
7. Confirm the default branch is `main`.
8. Update any local remote if needed:
   `git remote set-url origin https://github.com/FaramarzKowsari/computational-pathology-ai-lab.git`
