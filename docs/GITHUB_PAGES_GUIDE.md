# GitHub Pages

1. Push the complete repository.
2. Open **Settings > Pages**.
3. Under Build and deployment, choose **GitHub Actions**.
4. Run the `Deploy GitHub Pages` workflow or push a change inside `website/`.
5. The expected address is `https://FaramarzKowsari.github.io/computational-pathology-ai-lab/`.
6. Open the site and verify the book DOI, Google Books key, canonical URL, robots file, and sitemap.
7. Add the Pages URL to the repository About panel.
8. For Google Search Console, verify the Pages property and submit `/sitemap.xml` after deployment.
