# ابتدا این فایل را بخوانید

این بسته برای جایگزینی و توسعهٔ مخزن فعلی شما آماده شده است.

## روش پیشنهادی

1. از پوشهٔ محلی مخزن فعلی یک نسخهٔ پشتیبان بگیرید.
2. محتوای این پوشه را روی پوشهٔ محلی مخزن فعلی کپی کنید.
3. فایل `PREPARE_WINDOWS.bat` را اجرا کنید. این فایل، نوت‌بوک‌ها و نتایج اصلی پروژهٔ دانشگاه Boulder را بدون تغییر به پوشهٔ Legacy منتقل می‌کند.
4. در GitHub Desktop تغییرات را بررسی و Commit کنید.
5. مخزن را در GitHub به `computational-pathology-ai-lab` تغییر نام دهید.
6. GitHub Pages را روی **GitHub Actions** قرار دهید.
7. پس از سبزشدن Workflowها، Release نسخهٔ `v1.0.0` را منتشر کنید.
8. برای نرم‌افزار DOI جداگانه بگیرید. DOI `10.5281/zenodo.21444837` فقط متعلق به کتاب است.
