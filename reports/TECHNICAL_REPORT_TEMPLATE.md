# Technical Report Template

## Abstract
State the executed question, data, methods, and evidence without clinical overclaiming.

## Data and provenance
Dataset version, license, checksum, inclusion criteria, label source, split unit, exclusions.

## Methods
Architectures, preprocessing, augmentation, optimization, seeds, hardware, and stopping rules.

## Results
Discrimination, calibration, confidence intervals, subgroup results, error analysis, domain shifts, latency, and memory.

## Limitations
Internal validity, external validity, clinical relevance, bias, and unresolved failure modes.

## Reproducibility
Commit, environment lock, configuration, predictions, logs, and artifact checksums.
