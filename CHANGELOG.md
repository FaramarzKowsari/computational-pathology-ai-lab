# Changelog

All notable changes are documented here.

## [1.0.1] - 2026-07-19

### Changed
- Added the Zenodo Concept DOI for all software versions.
- Preserved the archived `v1.0.0` version-specific DOI.
- Unified README, GitHub Pages, citation, CodeMeta, and Zenodo metadata.
- Added the stable author biography, consistent author image, and official profile links.
- Expanded `Person`, `WebPage`, `SoftwareSourceCode`, and `Book` structured data.
- Updated package and source version metadata to `1.0.1`.
- Added release documentation for the metadata and discoverability patch.

### Scientific status
No clinical validation, external benchmark, foundation-model score, or diagnostic claim is added by this release.

## [1.0.0] - 2026-07-19

### Added
- Production-oriented Python package for patch-level computational pathology.
- CNN, transfer-learning, Vision Transformer, foundation-adapter, and MIL modules.
- Reproducible synthetic-data workflow and CPU smoke training.
- Medical-AI metrics, calibration, bootstrap confidence intervals, explainability, uncertainty, and drift utilities.
- FastAPI service, Streamlit demonstration, Docker assets, CI, DVC, and MLflow hooks.
- Model Card, Data Card, research documentation, GitHub Pages site, and companion-book metadata.
- Legacy migration scripts for the original University of Colorado Boulder assignment.

### Scientific status
No clinical validation, external benchmark, foundation-model score, or diagnostic claim is included. All example outputs are explicitly marked as synthetic or not yet benchmarked.
